Portfolio

These assignments were built for IT 312 at Central Washington University. Each one covers a different layer of designing digital environments from thinking in systems, to designing what users see, to building and prototyping the thing itself.

  • Blog Six posts on systems thinking, SDLC models, human-centered design, cognitive load, visual perception, and interaction ethics.
  • Design Challenge 1 - Modeling a Digital System Context-aware digital system diagrams built in FigJam for a smart thermostat environment.
  • Design Challenge 2 - Annotated Style Guide Annotated web app style guide built in Figma. Typography, color, spacing, and component standards for the smart thermostat interface.
  • Design Challenge 3 - Building Meaningful Interactions Design file for the smart thermostat interaction model built in Figma.
  • Design Challenge 3 - Clickable Prototype Interactive prototype demonstrating the user experience of the smart thermostat system.
